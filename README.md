# 💰 Gestor de Economia Doméstica

> **Status do Projeto:** 🚀 Finalizado (Entrega Inicial - Bootcamp 2)

## 🎯 Sobre o Projeto
Este projeto foi desenvolvido para resolver uma dor real: a **dificuldade de controle financeiro imediato**. Muitas pessoas não sabem quanto podem gastar ou se bateram suas metas de economia ao final do mês. 

Esta aplicação simples em Python permite que o usuário insira sua renda, despesas e meta, recebendo um feedback instantâneo sobre sua saúde financeira.

## 👥 Público-alvo
* Estudantes universitários.
* Microempreendedores que precisam de cálculos rápidos.
* Pessoas que buscam organização financeira pessoal.

## 🛠️ Tecnologias Utilizadas
* **Linguagem:** Python 3.13
* **Ambiente:** VS Code
* **Versionamento:** Git & GitHub
* **Automação (CI):** GitHub Actions
* **Testes:** Unittest (Nativo do Python)
